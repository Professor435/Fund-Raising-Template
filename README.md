# Fund-Raising-Template

This project is a responsive charity/donation website template built using HTML, CSS, Bootstrap, and JavaScript. It includes multiple pages such as Home, About, Mission, Donate, News, and Contact. The design is clean and user-friendly, with animations, image sections, and responsive layouts to work smoothly on desktop and mobile devices. All styling and interactivity are handled on the frontend, with no backend or database required.
